Kumpulan Codingan Penting & Fungsinya

1. Membaca Data dari Database (PHP MySQLi)

Mengambil semua data dari tabel users dan menampilkannya.

<?php
$conn = new mysqli("localhost", "root", "", "stuarz");
$result = $conn->query("SELECT * FROM users");
while($row = $result->fetch_assoc()) {
    echo $row['name'] . "<br>";
}
?>

2. Validasi Form HTML

Form dengan validasi wajib diisi dan email valid.

<form method="post">
    Nama: <input type="text" name="nama" required><br>
    Email: <input type="email" name="email" required><br>
    <button type="submit">Kirim</button>
</form>

3. AJAX Request (Fetch API)

Mengirim data ke server tanpa reload halaman.

<script>
fetch('/api/data', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({id: 1})
})
.then(res => res.json())
.then(data => console.log(data));
</script>

4. Membuat Tabel Responsive dengan CSS

Tabel tetap rapi di layar kecil.

<style>
table { width: 100%; border-collapse: collapse; }
th, td { border: 1px solid #ccc; padding: 8px; }
@media (max-width:600px) {
    table, thead, tbody, th, td, tr { display: block; }
    th { background: #eee; }
}
</style>

5. Membatasi Akses Halaman dengan Session (PHP)

Redirect jika user belum login.

<?php
session_start();
if (!isset($_SESSION['user_id'])) {
    header('Location: login.php');
    exit;
}
?>

6. Menampilkan Data Dinamis dengan Loop (PHP)

Menampilkan daftar nama dari array.

<?php
$nama = ['Andi', 'Budi', 'Cici'];
foreach ($nama as $n) {
    echo "<li>$n</li>";
}
?>

7. Membuat Modal Sederhana (HTML + CSS + JS)

Popup modal untuk konfirmasi atau info.

<button onclick="document.getElementById('modal').style.display='block'">Buka Modal</button>
<div id="modal" style="display:none;position:fixed;top:0;left:0;width:100%;height:100%;background:#0008;">
  <div style="background:#fff;padding:2em;margin:10% auto;width:300px;">

  </div>
</div>

8. Fungsi JavaScript Validasi Angka

Cek apakah input hanya angka.

function hanyaAngka(evt) {
    var charCode = (evt.which) ? evt.which : evt.keyCode;
    if (charCode < 48 || charCode > 57) return false;
    return true;
}

9. Membuat Redirect Otomatis (HTML Meta)

Redirect ke halaman lain setelah 3 detik.

<meta http-equiv="refresh" content="3;url=dashboard.html">
